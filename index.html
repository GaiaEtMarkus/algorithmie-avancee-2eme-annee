🧩 Pourquoi Tetris ?

Tetris combine parfaitement tous les concepts vus dans les 3 sessions :

  • ✓ Session 1 : Variables, fonctions, boucles, conditions
  • ✓ Session 2 : Tableaux 2D, structures de données, récursion
  • ✓ Session 3 : Optimisation, gestion des performances

🎯 Défis techniques

Structures

  • • Grille 2D
  • • Pièces rotatives
  • • Détection collisions

Algorithmes

  • • Détection lignes
  • • Suppression/décalage
  • • Gestion temps réel

🎯 Mission

Créer un Tetris complet from scratch en utilisant uniquement les concepts appris dans les 3 sessions.

✅ Exemples de Commits Corrects

Format : type: description
feat: implementation de la classe Tetris
feat: ajout de la détection de collisions
fix: correction du bug de rotation
refactor: optimisation de la boucle de jeu
docs: ajout des commentaires JSDoc
test: ajout des tests unitaires
Types de commits :
  • • feat: nouvelle fonctionnalité
  • • fix: correction de bug
  • • refactor: refactoring du code
  • • docs: documentation
  • • test: ajout de tests

❌ Exemple Éliminatoire

Repository avec seulement 2 commits :

commit 2: "fin du projet"
